# Decodelabs Cloud Computing Internship

This repository contains the projects and hands-on exercises completed during the Decodelabs Cloud Computing Internship. The internship focuses on AWS cloud services, infrastructure management, databases, serverless computing, and real-world cloud deployment practices.

---

## Internship Overview

Throughout this internship, I gained practical experience in:

- AWS Cloud Services
- Linux Server Administration
- Static Website Hosting
- Relational Database Management
- Serverless Computing
- Cloud Security Fundamentals
- Cloud Monitoring and Logging

## Tasks Completed

### Task 1 - Static Portfolio Hosting on AWS S3

#### Objective
Deploy a personal portfolio website using Amazon S3 Static Website Hosting.

#### Services Used
- Amazon S3
- Static Website Hosting

#### Key Activities
- Created an S3 bucket
- Uploaded portfolio website files

- Generated public website endpoint

### Task 4 - Serverless Cost Calculator

#### Objective
Build a serverless application using AWS Lambda.

#### Services Used
- AWS Lambda
- Amazon CloudWatch

#### Programming Language
- Python

#### Functionality
Accepts two numbers as input and returns their sum.

#### Sample Input

```json
{
  "num1": 15,
  "num2": 25
}
```

#### Sample Output

```json
{
  "Sum": 40
}
```

#### Skills Learned
- Serverless Computing
- Event-Driven Architecture
- AWS Lambda Functions
- CloudWatch Monitoring
- Python for Cloud Applications
